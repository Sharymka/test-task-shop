# Test Task Shop

Тестовое задание - карточный каталог с поиском и фильтрами.

## Технологии

- HTML
- JavaScript (vanilla)
- SCSS

## Локальный запуск

Просто откройте `index.html` в браузере или используйте любой локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server
```
